Introduction

題目：Steam Games Total Sales Analysis

透過各項數據（遊戲類型、遊戲網站評分、價格、支援搖桿、支援多個語言等等）預測該遊戲的擁有人數（銷量），及分析影響銷量的幾項重要feature

使用了以下的幾個預測模型：

  • Decision Tree
  • Random Forest
  • KNN
  • ANN
  • Linear Regression

由於預測的target為numeric，除了Linear Regression以外的Model皆會使用Binning後的data（分成數量同等份的五個類別）
除此之外，因為feature數量很多，也會嘗試使用PCA降維，並測試其performance如何

Preprocessing

Data Clean-up

在進行training前，要先檢視data的狀況，由於data的來源是利用爬蟲得來，可能會有格式跑掉或空格的data。
此外，不是每項row和feature都是我需要的，所以在稍微過目後我手動刪除了幾項的row和column：

  • row
    • 有缺項
    • 格式錯誤（由data type判斷）
    • GenreIsNonGame == True ： 該項不是遊戲 --> 題目僅分析遊戲類別
    • SteamSpyOwners == 0 ： 會導致model異常的noise
  • column
    • data type為文字、網址、email（i.e. AboutText：遊戲簡述、HeaderImage：遊戲封面的網址、SupportEmail：客服email…）
    • 與分析目標無關（i.e. DLCCount：DLC數量、AchievementCount：遊戲成就數量…）

Data binning

為了讓data能利用classification model來預測，將data切為數量同等份的五個bin

# convert string to int
games['SteamSpyOwners'] = games['SteamSpyOwners'].astype(int)

# cut numeric feature 'SteamSpyOwners' to 5 bins 
games['bin'] = pd.qcut(games['SteamSpyOwners'], q=5, precision=1)

Conclusion & Application

從上述的結果來看，這幾項model的表現都不是非常好，推測原因除了data本身target相當imbalance之外，亦有可能是大多feature與target沒有關聯，導致model很難從中找出target和feature的關係，進而無法產生好的預測。

題目希望透過各項feature預測該遊戲的擁有人數（銷量），可以用於遊戲發售前的各項事前預測，若遊戲項目為多人制線上遊戲，可依此預測發售後伺服器所需乘載的人數，也可依此調整販售價格達到最大利潤，亦可增加媒體和玩家的事前評測，提高媒體評分分數及玩家推薦數後推測能增加多少玩家購買意願。

亦可判斷一項投資是否能獲得預期收穫，如下列code例子：讓遊戲支援linux平台後能增加《刺客教條：兄弟會》這款遊戲約165k的玩家，若收支平衡後能獲得更多利潤，則可說這項投資是成功的。

test = (games[games['Name'] == 'Assassins Creed(r) Brotherhood'])
modified_test = test.drop(['Name', 'five_level', 'SteamSpyOwners'], axis = 1)
modified_test['PlatformLinux'] = True
a = regr.predict(modified_test)
print(a - test['SteamSpyOwners'].values[0])

此外，在經過decision tree的impurity計算後，我們可以得出影響銷量的前10大feature：

  • RecommendationCount
  • ReleaseYear
  • PriceFinal
  • Metacritic
  • SupportedLanguagesCount
  • IsFree
  • GenreIsEarlyAccess
  • GenreIsRPG
  • PlatformMac
  • CategoryMultiplayer

除了幾項與銷量有明顯相關的feature(ReleaseYear --> 累積時間越長勢必會有越多銷量、IsFree --> 免費的遊戲)

我們可以看見價格、媒體評測分數、支援的語言數量是影響銷量很大的原因，而玩家也偏好有early access（推測可由公開測試提高遊戲熱度，並透過讓玩家只體驗一部份遊戲提升對遊戲後續的好奇心，進而提升銷量）、RPG、多人類型的遊戲

而遊戲有沒有支援MAC也是影響遊戲銷量的一大因素
